# Bug Notes

Create one subdirectory per bug (name or ID).

Example:

- `batch-order-race-142/`
- `pcap-canonicalization-off-by-one/`

Inside each directory, add date-stamped markdown notes as debugging progresses.

## Note Template

### YYYY-MM-DD - <Short Title>

- Commit: <git hash or NO_COMMIT_YET>
- Symptom:
- Root cause:
- Fix:
- Risks/Tradeoffs:
- Validation/Test follow-up:
